## Enunciado

Leia um número inteiro e exiba "Negativo", "Zero" ou "Positivo", conforme o caso.

Ex.:
```
Digite um número: 35
Positivo
```
ou
```
Digite um número: -10
Negativo
```
ou
```
Digite um número: 0
Zero
```
